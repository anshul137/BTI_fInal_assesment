Reflection

Responsive design is the GUI (graphic user interface). We use this approach to create content which we can adjust to any screen size of the user. It is the browser reacting to the user’s device through media query that is CSS (Cascading Style Sheets). It detects the device and then serving up the friendly version of the resolution for that particular device such as standard desktop, mobile and mobile tablets. During my research I found that there are three main principles of responsive design. First, Fluid grid system which uses the relative measure and respond to the width of the viewport. Second, Fluid Image in which we shrink the size of the image that are on the web page according to the size of the users display screen. Lastly are the media queries which are the filter that we use so that we can detect the browsing device dimensions and make our design match with the user’s device screen size. Moreover, with the responsive design we can also hide certain items for example background images or the secondary content that is available on the page. Hiding content from the user depends upon the practical knowledge about the user and on which device he will be going to access the website.
The main potential advantage of responsive design is that we don’t need to develop different websites for the different device type. We prioritize the content of the website so that it becomes easier for the user so that they can find what they are looking for easily.
